# Dicas para os Calouros

## 1. Não deixe trabalhos para última hora
Na faculdade muitos trabalhos acumulam, então organização é essencial.

## 2. Pratique programação
Quanto mais você praticar lógica e programação, mais fácil as matérias vão ficar.

## 3. Participe das aulas
Prestar atenção nas aulas ajuda muito na hora das provas e trabalhos.

## 4. Use o GitHub
Aprender a usar o GitHub desde cedo ajuda muito no desenvolvimento de projetos.

## 5. Não deixe as horasa complementares pra depois
No curso existem horas complemetares e pra poder se formar devemos ter 240 horas complementeres completas.
